## 📌 Project Overview

This project presents an interactive Spotify Music Dashboard developed using Microsoft Power BI. The dashboard analyzes Spotify songs, artists, albums, genres, and popularity metrics through interactive visualizations. It enables users to explore music trends, identify top-performing artists, compare albums, and gain meaningful insights using dynamic filters and KPI cards.

---

## 🎯 Business Problem

Music streaming platforms generate large volumes of data. This dashboard helps visualize Spotify data to understand artist performance, song popularity, album trends, and genre distribution, enabling better data-driven insights.
